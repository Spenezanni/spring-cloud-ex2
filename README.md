# spring-cloud-ex2
Project spring-cloud-ex2

This is a Spring Boot microservice project, it's using Eureka-Server and Gateway.

![image](https://github.com/Spenezanni/spring-cloud-ex2/assets/32372447/728d708f-1a58-4d59-b3d9-fc4400381098)

Ms-Worker microservice call from Postman, return a list of Worker. The Url is composed of gateway port and calling directly ms-worker path from Eureka-Server.
